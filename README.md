# 姿态估计数据标注平台

这是一个用于姿态估计 (Pose Estimation) 任务的数据标注工具，基于 React 和 Node.js 开发，支持关键点和矩形框标注。

## 🚀 核心特性

- **高性能图库**：集成 `sharp` 库自动生成缩略图，支持数千张图片的秒级加载与流畅滚动。
- **精确标注**：支持矩形框 (BBox) 和 关键点 (Keypoints) 标注，并提供十字辅助线。
- **智能渲染**：标注框与图片加载状态严格同步，支持浏览器缩放实时自适应，彻底杜绝标注偏移。
- **姿态估计支持**：支持关键点分组、层级管理以及 YOLO 姿态数据集格式导出。
- **项目管理**：支持多项目管理、数据自动保存及批量导出。

## 🛠️ 技术栈

- **前端**：React, Lucide-React (图标), Vanilla CSS (现代透明玻璃感设计)
- **后端**：Node.js, Express, Multer (文件管理)
- **图像处理**：Sharp (高性能缩略图引擎)

## 📦 安装与启动

### 环境要求
- Node.js (推荐 v18+)

### 快速启动 (Windows)
双击根目录下的 `start.bat` 即可自动安装依赖并启动前后端服务。

### 手动安装
1. **安装后端依赖**：
   ```bash
   npm install
   ```
2. **安装前端依赖**：
   ```bash
   cd client
   npm install
   ```
3. **同时启动**：
   分别在根目录和 `client` 目录运行 `node server.js` 和 `npm run dev`。

## 📂 项目结构

- `/projects`：存储所有项目的图片、缩略图及标注 JSON 数据。
- `/client`：React 前端源代码。
- `server.js`：后端 API 服务核心代码。
- `.gitignore`：已配置好，自动过滤项目数据及依赖，方便直接上传 GitHub。

## ⚠️ 注意事项

- **数据安全**：项目数据默认存储在 `projects/` 下，已加入 `.gitignore`，不会被提交。
- **性能建议**：首次加载大量图片时，系统会生成缩略图，完成后速度会显著提升。
